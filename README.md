# Board de Tarefas

Este é um projeto de um sistema simples para gerenciar tarefas, com funcionalidades de adicionar, remover, listar e atualizar tarefas. O projeto utiliza Java, MySQL e JDBC.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Atualizar tarefas
- Remover tarefas

## Tecnologias utilizadas

- Java
- MySQL
- JDBC
